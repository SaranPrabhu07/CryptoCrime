Crypto Tools

📡 Investigation Resource: These are your weapons. Each level uses one of the techniques below. Study them. Use them when you're stuck. The clock doesn't stop for anyone.
ROT13
Substitution Cipher
Easy▼

ROT13 rotates each letter by 13 positions. A→N, B→O, etc. It's symmetric — encoding and decoding use the exact same operation.

Used in: Level 1 · Tip: Every letter maps to exactly one other letter.
Caesar Cipher
Shift Cipher
Easy▼

Shifts every letter by a fixed number. With shift 3: A→D, B→E. If you don't know the shift, use brute force to try all 25 options.

Used in: Level 2 (step 2) · Tip: Look for the shift that produces a real word or phrase.
Binary ↔ Text
Base-2 Encoding
Easy▼

Binary represents data as 0s and 1s. Each character is 8 bits. A = 01000001. Convert binary to ASCII to get readable text.

Used in: Level 2 (step 1) · Tip: Copy the output into the Caesar decoder for step 2.
Vigenère Cipher
Polyalphabetic Cipher
Medium▼

Uses a keyword to apply different Caesar shifts to each letter. The keyword repeats over the message. You need the exact keyword to decode it.

Used in: Level 4 · Tip: The keyword is hidden inside a riddle on the level page.
AES Decryption
Advanced Encryption Standard
Hard▼

AES is modern military-grade encryption. It uses a password (key) to encrypt and decrypt. Without the key, it's impossible to crack. The key is hidden in a riddle.

Use CyberChef to decrypt:
1. Recipe: AES Decrypt
2. Mode: ECB · Key format: UTF8
3. Paste encrypted string → enter key → Bake

Used in: Level 5 · Tip: The password will be a single lowercase word derived from the riddle.
Steganography
Hidden in Plain Sight
Medium▼

Hides secret data inside ordinary image files. The image looks completely normal. The message is encoded in the least significant bits of pixels — invisible without the right tool.

Steps to extract:
1. Download the image from the level page
2. Open the decoder tool below
3. Upload the image → click Decode
4. The hidden text will appear

Used in: Level 3 · Tip: The image looks perfectly ordinary. It isn't.
SHA-256 Hash
One-way Cryptographic Hash
Hard▼

SHA-256 converts any input into a fixed 64-character hash. It's one-way — you can't reverse it. Hash each suspect name and compare to find the match.
